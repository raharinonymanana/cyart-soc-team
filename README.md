# cyart-soc-team

This repository contains my Week 2 SOC task submission.

## Repository Purpose
The goal of this submission is to demonstrate both theoretical understanding and practical application of Security Operations Center (SOC) concepts, including alert prioritization, incident classification, triage, response documentation, evidence preservation, and a full alert-to-response workflow.

## Folder Overview
- Week 2/01_Theory — theoretical notes and PDF documents on core SOC concepts
- Week 2/02_Alert_Management — alert classification, prioritization, dashboard output, incident ticket, and escalation email
- Week 2/03_Response_Documentation — incident response template, investigation timeline, checklist, and post-mortem
- Week 2/04_Alert_Triage — triage examples and IOC validation
- Week 2/05_Evidence_Preservation — evidence collection and chain-of-custody documentation
- Week 2/06_Capstone — full alert-to-response cycle report and stakeholder briefing
- Week 2/07_Screenshots — supporting screenshots from the practical work
- Week 2/08_Workflow — step-by-step workflow followed during the task
